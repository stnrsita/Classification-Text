Title: Kereta Api Tabrak Truk Mogok di Sergai, Masinis Alami Luka Bagian Kaki

Content:
Kecelakaan melibatkan kereta api dengan truk terjadi di perlintasan sebidang di Jalan Lintas Sumatera tepatnya di Pasar Bengkel, Kecamatan Perbaungan, Kabupaten Serdang Bedagai (Sergai). Akibatnya masinis mengalami luka pada bagian kaki.
Dilansir detikSumut, dalam video yang dilihat Selasa (19/3/2024), sebuah truk terlihat melintang di perlintasan sebidang kereta api. Palang perlintasan kereta api tersebut juga terlihat sudah diturunkan dan truk berada di tengah-tengah.
Sesaat kemudian, kereta api melintas dan menabrak truk tersebut. Truk terlihat terseret oleh kereta api. Ps Kasi Humas Polres Sergai Iptu Edward Sidauruk membenarkan kecelakaan itu terjadi sekitar pukul 21.00 WIB.
"Benar ada kecelakaan, sekitar pukul 21.00 WIB tadi," kata Iptu Edward Sidauruk kepada detikSumut.
Edward menjelaskan jika truk tersebut mengalami kerusakan saat di perlintasan. Truk tersebut ditinggalkan begitu saja oleh supir dan kernet nya.
Sementara asisten masinis mengalami luka di bagian kaki dan dibawa ke rumah sakit terdekat. Kereta api itu sendiri disebutkan melaju dari arah Kota Tebing Tinggi menuju Kota Medan.
"Masinisnya mengalami luka kaki, jadi udah dibawa untuk berobat ke rumah sakit terdekat," ucapnya.
