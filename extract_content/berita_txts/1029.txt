Title: Gedung Hubdam Kodam Diponegoro Semarang Sempat Terbakar

Content:
Sebuah gedung di komplek Kodam IV/Diponegoro, Semarang terbakar siang ini. Api berhasil dipadamkan setelah 2 jam pemadaman.
Kapendam Kodam IV/Diponegoro Kolonel Inf Richard Harison menyebut kebakaran terjadi di Gedung Perhubungan Daerah Militer atau Hubdam sekitar pukul 11.30 WIB. Kemudian api padam sekitar pukul 13.30 WIB.
"Sudah, 13.30 WIB sudah padam," katanya saat dikonfirmasi Jumat (29/3/2024).
"Jadi itu kejadiannya kan sedang sepi itu ada beberapa anggota dari Hubdam itu kan sedang curving melihat asap atau api lah di atas akhirnya dia naik," jelasnya.
Para anggota itu sempat mencoba memadamkan api secara mandiri. Karena api terlanjur besar, akhirnya mereka fokus menyelamatkan barang-barang di lokasi dan memanggil Damkar Kota Semarang.
"Sempat mencoba memadamkan juga tapi api sudah besar, tidak sanggup akhirnya mereka mengamankan barang-barang yang bisa diselamatkan termasuk juga menelepon dari Dinas Pemadam Kebakaran alhamdulillah ada enam unit yang datang dan bisa dipadamkan," ujarnya.
Ada beberapa ruangan di lantai 2 yang terbakar. Namun, peralatan di dalamnya disebut telah berhasil diselamatkan.
"Alat-alat perhubungan aman, ruangan aja, yang pasti itu kita lihatnya bagian atas lah satu atau berapa," tambahnya.
Meski begitu, pihaknya masih mendata terkait kerugian dari kebakaran tersebut. Termasuk mencari tahu penyebab dari kebakaran itu.
"Kerugian lengkapnya kita masih menghitung ya apa saja kira-kira yang terbakar. Untuk penyebabnya itu masih dalam penyelidikan ya," kata Richard.
