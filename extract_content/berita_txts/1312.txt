Title: Jakarta LavAni Allo Bank Raih Kemenangan Perdana di Proliga 2024

Content:
Jakarta LavAni Allo Bank berhasil meraih kemenangan perdana di Proliga 2024. Jawara bertahan Proliga itu berhasil mengalahkan Garuda Jaya tiga set langsung dengan skor 25-16, 25-22, 25-20.
Bermain di GOR Among Rogo Jogja, Kamis (25/4/2024), Jakarta LavAni Allo Bank mendominasi jalannya laga sejak set pertama. Mereka unggul cepat 3-0 dari Garuda Jaya di awal pertandingan.
Dua rekrutan asing LavAni, Mohammad Reza Beik dan Renan Buiatti tampil cukup gemilang. Mereka langsung nyetel dengan mencetak beberapa poin.
Jakarta LavAni Allo Bank berhasil unggul mudah atas Garuda Jaya di set pertama partai pembuka dengan skor 25-16.
Di set kedua, LavAni masih mendominasi. Sementara Garuda Jaya cukup tertekan namun bermain cukup merepotkan tim tuan rumah.
Sempat hampir terkejar pada poin 20-19, LavAni kembali memastikan keunggulan di set kedua atas Garuda Jaya dengan skor 25-22.
Di set ketiga, kedua tim bermain sangat kuat, perolehan poin juga ketat. Garuda Jaya unggul pertama kalinya di time out dengan poin 8-7.
Jakarta LavAni Allo Bank berhasil memetik poin penuh di laga perdana Proliga 2024 usai memastikan kemenangan di set ketiga dengan skor 25-20.
