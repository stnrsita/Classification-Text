3 Rumah di Cianjur Terbakar Usai Warga Tertidur Saat Masak untuk Buka Puasa

Ikbal Selamet - detikNews
Selasa, 26 Mar 2024 10:15 WIB
Jakarta -

Tiga rumah di Kampung Sayangheulang, Kelurahan Bojongherang, Kecamatan Cianjur, Kabupaten Cianjur, ludes terbakar. Kebakaran diduga disebabkan api dari tungku yang menyambar bangunan saat pemilik rumah yang tertidur saat memasak untuk berbuka puasa.

Dilansir detikJabar, peristiwa ini terjadi pada Senin (25/3/2024). Berdasarkan informasi yang dihimpun, api pertama kali terlihat dari rumah semipermanen milik Yeti Suryati.

Rumah yang terdiri dari bahan mudah terbakar membuat api cepat merembet dan membesar. Bahkan kobaran api juga menjalar ke dua rumah lain yang saling berdekatan di kawasan padat permukiman tersebut.

"Keterangan dari saksi, api terlihat pertama kali dari rumah Ibu Yeti. Kemudian merembet ke dua rumah lainnya. Jadi total ada tiga rumah yang terbakar," ujar Reki, salah seorang petugas pemadam kebakaran Cianjur, dilansir detikJabar.

Dia menambahkan, dari hasil penelusuran, diketahui kebakaran dipicu tungku api yang tidak terpantau oleh pemilik rumah. Akibatnya, api menjalar ke seluruh bagian rumah.

"Jadi penghuni rumah memasak di tungku untuk buka puasa. Tapi saat memasak itu ditinggal tidur. Sehingga api menjalar. Kalau kerugian akibat kebakaran ditaksir mencapai ratusan juta rupiah," tuturnya.
