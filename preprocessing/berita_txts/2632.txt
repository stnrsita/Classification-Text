 bangunan bale sakenem di tabanan ludes terbakar saat pemilik tertidur pulas bangunan bale sakenem milik warga di desa sembung gede kecamatan kerambitan tabanan bali ludes terbakar. bangunan tradisional khas bali berukuran 3x6 meter itu dilalap si jago merah saat pemilik bangunan sedang tertidur pulas. kapolsek kerambitan kompol i gusti putu sudura mengungkapkan peristiwa itu terjadi sekitar pukul 02.00 wita minggu 1442024. ia menegaskan tidak ada korban jiwa akibat kejadian tersebut. dugaan penyebab kebakaran karena korsleting listrik kata sudura melalui keterangannya minggu. sudura menjelaskan api pertama kali dilihat oleh seorang saksi bernama i nyoman antara 32. sontak pria itu lantas berteriak dan membangunkan pemilik bangunan bernama i ketut suanda 61 yang sedang tertidur. menurut sudura kobaran api pertama kali muncul dari dalam bangunan. antara bersama warga sekitar sempat membantu memadamkan api dengan peralatan seadanya. tak lama kemudian petugas pemadam kebakaran damkar tabanan datang dengan mengerahkan dua unit mobil damkar. api berhasil dipadamkan sekitar pukul 02.55 wita. kebakaran sudah ditangani dini hari tadi oleh damkar tabanan. untuk kerugian ditaksir rp 90 juta pungkasnya.
