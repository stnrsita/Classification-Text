 gedung hubdam kodam diponegoro semarang sempat terbakar sebuah gedung di komplek kodam ivdiponegoro semarang terbakar siang ini. api berhasil dipadamkan setelah 2 jam pemadaman. kapendam kodam ivdiponegoro kolonel inf richard harison menyebut kebakaran terjadi di gedung perhubungan daerah militer atau hubdam sekitar pukul 11.30 wib. kemudian api padam sekitar pukul 13.30 wib. sudah 13.30 wib sudah padam katanya saat dikonfirmasi jumat 2932024. jadi itu kejadiannya kan sedang sepi itu ada beberapa anggota dari hubdam itu kan sedang curving melihat asap atau api lah di atas akhirnya dia naik jelasnya. para anggota itu sempat mencoba memadamkan api secara mandiri. karena api terlanjur besar akhirnya mereka fokus menyelamatkan barangbarang di lokasi dan memanggil damkar kota semarang. sempat mencoba memadamkan juga tapi api sudah besar tidak sanggup akhirnya mereka mengamankan barangbarang yang bisa diselamatkan termasuk juga menelepon dari dinas pemadam kebakaran alhamdulillah ada enam unit yang datang dan bisa dipadamkan ujarnya. ada beberapa ruangan di lantai 2 yang terbakar. namun peralatan di dalamnya disebut telah berhasil diselamatkan. alatalat perhubungan aman ruangan aja yang pasti itu kita lihatnya bagian atas lah satu atau berapa tambahnya. meski begitu pihaknya masih mendata terkait kerugian dari kebakaran tersebut. termasuk mencari tahu penyebab dari kebakaran itu. kerugian lengkapnya kita masih menghitung ya apa saja kirakira yang terbakar. untuk penyebabnya itu masih dalam penyelidikan ya kata richard.
