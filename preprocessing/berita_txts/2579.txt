 tertidur saat masak untuk buka puasa 3 rumah di cianjur terbakar tiga rumah di kampung sayangheulang kelurahan bojongherang kecamatan cianjur kabupaten cianjur ludes terbakar senin 2532024. kebakaran diduga disebabkan api dari tungku yang menyambar bangunan saat pemilik rumah memasak untuk berbuka puasa. informasi yang dihimpun detikjabar api pertama kali terlihat dari rumah semi permanen milik yeti suryati. rumah yang terdiri dari bahan mudah terbakar membuat api cepat merembet dan membesar. bahkan kobaran api juga menjalar ke dua rumah lain yang saling berdekatan di kawasan padat permukiman tersebut. keterangan dari saksi api terlihat pertama kali dari rumah ibu yeti. kemudian merembet ke dua rumah lainnya. jadi total ada tiga rumah yang terbakar ujar reki salah seorang petugas pemadam kebakaran cianjur. beruntung penghuni rumah berhasil menyelamatkan diri sesaat sebelum api membesar. korban tidak ada yang di dalam rumah berhasil keluar dan menyelamatkan diri kata dia. reki menjelaskan api akhirnya berhasil dipadamkan dua jam setelah kejadian dengan menerjunkan empat unit mobil pemadam dan 30 personel. kejadian jam 18.00 wib dan berhasil dipadamkan sekitar pukul 20.00 wib. penanganan cukup memakan waktu karena kebakaran terjadi di pemukiman padat dan masuk ke dalam gang sempit tuturnya. dia menambahkan dari hasil penelusuran diketahui kebakaran dipicu tungku api yang tidak terpantau oleh pemilik rumah. akibatnya api menjalar ke seluruh bagian rumah. jadi penghuni rumah memasak di tungku untuk buka puasa. tapi saat memasak itu ditinggal tidur. sehingga api menjalar. kalau kerugian akibat kebakaran ditaksir mencapai ratusan juta rupiah pungkasnya.
